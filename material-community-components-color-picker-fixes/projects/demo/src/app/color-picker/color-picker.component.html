Overview

MccColorPicker

When import the MccColorPickerModule, you can provide some configurations.

 import {{ '{' }} MccColorPickerModule {{ '}' }} from 'material-community-components/color-picker';

  @NgModule({{ '{' }}
    ...
    imports: [
      MccColorPickerModule.forRoot({{ '{' }}
        empty_color: 'transparent',
        used_colors: ['#000000', '#FFF555']
        selected_icon: 'check'
        disable_selected_icon: false
        enable_alpha_selector: false
      {{ '}' }})
    ]
    ...
  {{ '}' }})

Configurations

Name Description Default
empty_color: string Define any color you want none
used_colors: string[] Define list of colors to start used colors []
selected_icon: string Define icon to be used on selected colors done
selected_svg_icon: string Define SVG icon name to be used on selected colors (nees to be registered in MatIconRegistry). Overrides "selected_icon" null
disable_selected_icon: boolean Disable selected icon showed when color is selected false
enable_alpha_selector: boolean Enable alpha selector option false

For @Input and @Output of the component, check API tab.

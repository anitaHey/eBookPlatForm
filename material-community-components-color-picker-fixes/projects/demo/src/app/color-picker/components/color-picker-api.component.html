Components

MccColorPicker

This is the main class of the component.

Inputs

Name Description
disabled: boolean Disable color-picker pop up
alpha: boolean Enable alpha selector, default: false
format: ColorStringFormat Select color string format, default: 'hex'
usedColorLabel: string Change the label of the used colors collection, default: 'USED COLORS'
usedColorStart: string[] Set initial value for used colors collection
reverseUsedColors: boolean Reverse order of used colors list, default: false
usedColorsPosition: MccUsedColorsPosition Set position for used colors collection to be show, default: 'top'
hideHexForms: boolean Hide the hexadecimal color input form, default: false
hideEmpty: boolean Hide all empty slots of the default collection, default: false
hideTransparentUsedColors: boolean Hide transparent option of UsedColors, default: false
hideUsedColors: boolean Hide UsedColors default collection, default: false
hideButtons: boolean Hide the buttons (confirm/cancel), default: false
hideColorPickerSelector: boolean Hide the color picker selector, default: false
colorPickerSelectorHeight: number Define new height of selector area, default: 170
selectedColor: string Define initially selected color
isOpen: boolean Define if panel will initiate open
overlay: boolean Define if panel will be show inside overlay, default: true
usedSizeColors: number Set the size of used colors collection, default: 30
btnCancelLabel: string Will replace the default text on cancel button, default: 'CANCEL'
btnConfirmLabel: string Will replace the default text on confirm button, default: 'CONFIRM'

Outputs

Name Description
change: EventEmitter Event emitted when the color changed, emits color string
selected: EventEmitter Event emitted when the color is selected, emits color string
clickOut: EventEmitter Event emitted when is user clicks outside of the component, emits nothing
canceled: EventEmitter Event emitted when is user clicks on cancel button, emits nothing

MccColorPickerCollection

This class is used to pass your own collection of colors to be displayed inside the color picker.

Inputs

Name Description
hideEmpty: boolean Hide all empty slots inside de collection
label: string String with the name of the collection
colors: MccColorPickerOption[] Array of the strings or MccColorPickerItem to be displayed inside the collection
size: number Limit of colors to be displayed, default: 30
transparent: boolean Show transparent options, default: false)

Outputs

Name Description
changeColor Event that id emitted when a color is clicked

Services

MccColorPickerService

You can use service to add, get and reset the used colors component data.

Methods

Name Description
addColor(color: string): void Add new color to the used colors collection
getUsedColors(): Observable{{ '<' }}string[]{{ '>' }} Return observable of used colors collection
resetUsedColors(): void Clear the used colors collection

Directives

Using MccColorPickerOrigin and MccConnectedColorPicker, you can connect the color picker with any input, textarea or select element.

MccColorPickerOrigin

This directive goes on the element (input, textarea or select) that will be connected to the color picker.

Inputs

Name Description
mccColorPickerOrigin: 'openOnFocus' | 'default' Enable opening color picker on focus, default: 'default'

Outputs

Name Description
changes: string Emit changes from the origin

MccConnectedColorPicker

This directive is used on the color picker that should be connected to the input

Inputs

Name Description
mccConnectedColorPickerOrigin: MccColorPickerOrigin Element that color picker connects to

Interfaces & Types

ColorStringFormat

Is a type created to handle colors. This type accept hex, rgb and hsl.

MccUsedColorsPosition

Is a type created to handle used colors position. This type accept the values top and bottom.

MccColorPickerOption

Is a type created to handle color collections. This type accept string or MccColorPickerItem.

MccColorPickerItem

This interface is used when you have the name of the color. This can help screen readers.

Attribute Description
text: string Name of the color
value: string Hexadecimal of the color

Components

MccSpeedDial

This is the main class of the component.

Inputs

Name Description
open: boolean Set initial open state of the component
mouseHover: boolean When enabled, handle open/closed state on mouse hover
spin: boolean Enable/disable spin animation
direction: DIRECTION Define direction of the actions button

Outputs

Name Description
openStateChange: EventEmitter Event emitted when the open state change

MccSpeedDialActions

This class is used to pass action buttons that will be showed when fab speed dial is opened.

Inputs

Name Description
animation: ANIMATION Set type of animation that will be executed on open/close

Types

DIRECTION up | down | left | right
ANIMATION scale | fling
